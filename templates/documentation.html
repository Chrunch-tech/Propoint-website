Well to use command palette you just need to type commnd that you need to execute for example: Lets say you need to add column A and B then you simply have to type A + B and this automatically generate a new column for you and if you want to add column A and B and put it equal to another column C then you just simply have to type C = A + B and this will put C equal to the sum of A and B

Merge conflict

In this section we'll talk about merge conflict error. What is it and How to avoid it.


What is merge conflict?

Merge conflict is an error which is happen when two merging cell has conflicting difference for example let's say you have add two column A and B you enter the command in the command palette and you hit execute then what going to happen now that propoint will read all the cell in that columns and attempt to add them if cell a1 and b1 have not same values means a1 is equal to a word and b1 is equal to a number that's where this error is occour


How to avoid this error?

First of all make the selected column don't have conflicting changes but if it still happen then Propoint ask you to Overwrite and Don't Overwrite So what are this:

Overwrite - In this case Propoint will overwrite the target cell with a empty cell.
Don't overwrite - In this case Propoint don't overwrite the target cell it will put it equal to its value.
